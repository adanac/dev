create dev by jekyll

  • 新建根目录：mkdir dev
  • 切换到目录：cd dev
  • git初始化：git init
  • 创建无父节点分支gh-pages：git checkout --orphan gh-pages
  • 创建配置文件：vi _config.yml，内容为：baseurl: /dev
  • 创建模板文件：mkdir _layouts，进入该目录，创建一个default.html文件，作为Blog的默认模板
  • 创建文章：mkdir _post，进入该目录，创建文章即可。
  • 创建首页：回到根目录，创建一个index.html文件
  • 添加发布内容：git add .
  • 提交发布内容：git commit -m"u";
  • 登录github网站，创建一个名为dev的库
  • 将发布内容推送到github刚创建的库：git remote add origin https://github.com/adanac/jekyll_demo.git
  • 推送内容：git push origin gh-pages
